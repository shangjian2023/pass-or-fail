# 期末急救计算器 - First Aid Calculator

> 🚑 期末求生，一键计算！看看你需要考多少分才能活下来...

## 项目简介

期末急救计算器是一款专为大学生设计的期末成绩计算工具。通过输入平时分、设置权重和目标分数，快速计算出期末考至少需要多少分才能安全过关。

## 功能特点

### 🎯 核心功能
- **平时分输入**：数字键盘快速输入，支持0-100分
- **权重设置**：提供30%、40%、50%预设，支持自定义权重
- **目标设定**：安全通过(60分)、争取优秀(80分)、冲击满绩(90分)
- **智能计算**：一键计算期末所需最低分数

### ✨ 特色亮点
- **戏谑风格文案**：根据计算结果展示不同风格的趣味文案和颜文字
  - 🎉 稳了！已经稳了！(๑•̀ㅂ•́)و✧
  - 😌 问题不大，稳如老狗(｡•̀ᴗ-)✧
  - 💪 有点东西，但东西不多(；′⌒`)
  - 🚑 危！建议直接准备补考Σ(っ°Д°;)っ

- **漂浮动画效果**：页面元素带有优雅的漂浮动画，提升视觉体验
- **急救箱主题**：经典深红色急救箱配色方案
- **数字动画**：分数变化时带有平滑的数字滚动效果
- **音效反馈**：计算完成时播放提示音

### 🎨 视觉设计
- 经典急救箱深红色系配色
- 毛玻璃效果卡片设计
- 响应式布局，适配移动端
- 流畅的过渡动画

## 使用方法

1. **输入平时分**：使用数字键盘输入你的平时成绩
2. **选择权重**：点击选择平时分占总成绩的权重比例
3. **设定目标**：选择你期望的总评目标分数
4. **启动计算**：点击"🚑 启动急救计算"按钮
5. **查看结果**：根据结果制定复习计划

## 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **样式**：CSS Variables、Flexbox、Grid
- **动画**：CSS Keyframes、Transitions
